# Emitir Documentos BR

Use this skill to operate Burocrata through its built-in local scripts. The repository is public and open source, so keep the workflow generic, reproducible, and safe for external users.

## Quick start

Install the skill:

```bash
npx skills add guimaferreira/burocrata-skills --skill emitir-documentos-br
```

Authenticate the local helper for `burocrata.com.br` with a token generated by the user in `https://burocrata.com.br/cli`:

```bash
node scripts/burocrata.com.br.js login --token <token>
```

If anything fails before execution, run diagnostics first:

```bash
node scripts/burocrata.com.br.js doctor
```

## Workflow

1. Identify the requested document in natural language.
2. Run `whoami` or `doctor` before searching if auth or connectivity may be an issue.
3. If the exact service is unclear, search first.
4. Inspect required fields before asking follow-up questions.
5. Ask only for missing required fields.
6. Check credits when needed.
7. Execute once all required fields are available.
8. Summarize the result clearly.

## Local script pattern

Preferred commands from the skill directory:

```bash
node scripts/burocrata.com.br.js search "<user request>" --json
node scripts/burocrata.com.br.js schema <tool-name> --json
node scripts/burocrata.com.br.js credits --json
node scripts/burocrata.com.br.js run <tool-name> --input '{"field":"value"}' --json
```

Use `node scripts/burocrata.com.br.js whoami --json` to confirm the current authenticated user.
Use `node scripts/burocrata.com.br.js doctor --json` to diagnose bad config, missing token, or broken connectivity.

## Guardrails

- Stay restricted to Brazilian document and certificate workflows supported by Burocrata.
- Never invent fields or unsupported services.
- Never ask again for a value already present in the conversation.
- Never browse the web to discover a tool or scrape the site when the local helper fails. Run `doctor`, then ask the user to fix token/auth if needed.
- If the user writes an obvious typo such as "DND Federal", infer the likely intent as "CND Federal" and confirm briefly.
- If credits are insufficient, stop and tell the user to recharge in the Burocrata app.
- If authentication fails, ask the user to re-authenticate.
